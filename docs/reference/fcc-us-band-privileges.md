# US FCC Amateur Radio Band Privileges

Source: ARRL US Amateur Bands frequency allocations page
URL: https://www.arrl.org/frequency-allocations
Regulation: 47 CFR Part 97, Subpart D, §97.301 (authorized frequency bands) and §97.305 (authorized emissions)

---

## License Classes

| Class       | Notes |
|-------------|-------|
| Novice      | Legacy class; no new licenses issued since 2000 |
| Technician  | Entry level; limited HF, full VHF/UHF |
| General     | Mid-level; broad HF access with some segments reserved for Extra |
| Advanced    | Legacy class; no new licenses issued since April 2000 |
| Amateur Extra | Full privileges on all amateur bands |

## Key Differences: ADIF Band Edges vs. US General Class Privileges

The ADIF band table uses the **total amateur allocation edge-to-edge**. A US General
class operator does **not** have access to the full ADIF band range on several HF bands:

| Band | ADIF edge (MHz) | General class can use (MHz) | Missing segments |
|------|-----------------|----------------------------|-----------------|
| 80m  | 3.500–4.000 | 3.525–3.600, 3.800–4.000 | 3.500–3.525 (Extra only); 3.600–3.800 (Extra/Adv) |
| 40m  | 7.000–7.300 | 7.025–7.125, 7.175–7.300 | 7.000–7.025 (Extra); 7.125–7.175 (Extra/Adv) |
| 20m  | 14.000–14.350 | 14.025–14.150, 14.225–14.350 | 14.000–14.025 (Extra); 14.150–14.225 (Extra/Adv) |
| 15m  | 21.000–21.450 | 21.025–21.200, 21.275–21.450 | 21.000–21.025 (Extra); 21.200–21.275 (Extra/Adv) |
| 60m  | 5.060–5.450  | 5 discrete channels only | Not a contiguous range |

All other bands in `Band::from_frequency_khz` (160m, 30m, 17m, 12m, 10m, 6m, 2m, 70cm)
are **full-band for General class** — ADIF edges and US General privileges align.

## Implementation Notes (future "privilege check" feature)

A US General class privilege check cannot be a single range per band. It requires:

1. Map the frequency to an ADIF band (existing `Band::from_frequency_khz`).
2. Check the frequency against General class sub-ranges for that band.
3. Optionally check mode restrictions (e.g., no phone on 30m; CW-only segments on lower edge of several bands).

The 80m, 40m, 20m, and 15m bands each have **two disjoint General sub-ranges** with
a gap in between (reserved for Extra/Advanced). 60m is channelized with 5 specific
center frequencies.

Advanced class is a legacy class (no new licenses since April 2000) and can be treated
the same as Extra for practical purposes in a privilege-check UI.
